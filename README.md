### Python To-Do List Application

This is a simple console-based To-Do List application built using Python. It allows users to add tasks, view all tasks, and remove completed or unwanted tasks through a menu-driven interface.

### Features

Add new tasks to the to-do list

View all existing tasks with numbering

Remove tasks by selecting their task number

Simple and beginner-friendly console interface

Technologies Used

Python (Core concepts)

Concepts Covered

Lists

Loops

Conditional statements

Functions

User input handling

How to Run the Project

Make sure Python is installed on your system.

Copy the code into a file named todo.py.

Open a terminal or command prompt.

Run the program using:

python todo.py


Follow the on-screen menu to manage your tasks.

Use Case

This project is ideal for beginners who want to practice Python fundamentals and build a small real-world application using basic logic.
